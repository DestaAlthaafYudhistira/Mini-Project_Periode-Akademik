## 🎨 Fitur-Fitur Utama

### 1. **Navigation Bar**
- Menu links: Dashboard, Periode Akademik, Program Studi, Kurikulum, Mahasiswa, Setting
- Responsive mobile menu dengan hamburger button
- Blue color scheme dengan hover effects

### 2. **Halaman Periode Akademik**
- **Breadcrumb Navigation**: Beranda > Setting > Periode Akademik
- **Search Bar**: Real-time search by kode atau nama periode
- **Action Buttons**:
  - "Tambah" - Open modal untuk add data
  - "Hapus" - Delete selected rows

### 3. **Tabel Periode Akademik**
- **Kolom**: Kode, Nama Periode, Tgl. Awal Kuliah, Tgl. Akhir Kuliah, UTS, UAS, Total Program, Status, Aksi
- **Data Sample**: 10 perioda akademik
- **Bulk Selection**: Checkbox untuk select/deselect rows
- **Edit Action**: Button untuk edit setiap row

### 4. **Search & Filter**
- Real-time search functionality
- Filter dropdown
- Auto reset pagination

### 5. **Pagination**
- Previous/Next buttons
- Page number indicators
- Dropdown untuk jumlah data per halaman
- Info: "Hal X/Y (Z data, time)"

### 6. **Modal Form**
- Input fields: Kode, Nama Periode, Tanggal (date picker)
- Validasi form
- Tombol Batal & Simpan
- Auto-close setelah submit

### 7. **Responsive Design**
- Mobile-first approach
- Hamburger menu untuk mobile
- Responsive table dengan overflow
- Optimized untuk semua ukuran layar
